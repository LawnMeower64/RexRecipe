# RexRecipe
Very small UI coded in python for training purposes. It tracks the progress of recipes on Rex:R


### Usage
Launch the program, take screenshots of your pinned recipe, paste it *(ctrl + v)* into the app, press enter then the result is automatically copied in your clipboard. You can also press backspace to remove all the screenshots.


### Installation
You might need to install [**EasyOCR**](https://github.com/JaidedAI/EasyOCR) through pip:
```
pip install easyocr
```

Also, this code works **ONLY** for **Windows**. I haven't tried neither MacOS nor Linux distros but I am pretty confident it will NOT work on those.


### Run
Use the python 3.13 Interpreter. I could then use the [**PIL**](https://github.com/python-pillow/Pillow/) libraries that came with my installed python version.
I'm pretty sure any python version above 3.13 works if you install PIL.
